## Aim

To understand and implement AWS Identity and Access Management (IAM) by
exploring users, groups, policies, permissions, and secure access
control.

## Objectives

-   Explore the IAM Dashboard.
-   Examine IAM Users and Groups.
-   Review IAM policies.
-   Compare Read-Only and Administrative permissions.
-   Assign users to IAM groups.
-   Verify group memberships.
-   Test IAM user permissions.
